O CADASTRO DE MEDICAMENTOS

   No cadastro de medicamentos registram-se todos os medicamentos que serão usados nas operações de estoque. Sempre que houver a necessidade de se colocar algum medicamento no estoque, deve-se primeiramente cadastra-lo nesta função.

A TELA DE CADASTRO DE MEDICAMENTOS   

   Oriente-se pelos números exibidos na figura para visualizar suas funções explicadas:

  1. Para procurar dinâmico e rapidamente as informações desejadas, digite nesta caixa o nome do procurado, a cada letra que você digitar os dados serão filtrados aos nomes que mais se assemelham ao digitado.
  2. Nesta área aparecem todos os cadastros efetuados, seja usando o filtro ou não. Clique sobre o nome desejado e carregue suas informações para o formulário, assim tenha as opções de excluir e alterar seus dados. 
  3. Nesta barra de tarefas suspensa aparecerão e desaparecerão conforme a necessidade os botões que salvam, excluem ou alteram os dados do cadastro.
  4. Aqui é exibido informações inalteráveis do sistema, como CÓDIGO, DATA E HORA DE CADASTRAMENTO.
  5. Neste formulário de informações serão exibidos os dados procurados. Para ver clique sobre seu nome na área 2.
  6. Clique sobre este link e obtenha um relatório com toda relação de dados cadastrados no sistema.
