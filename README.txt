SICEP - Sistema para Controle de Estoque e Pacientes
Software desenvolvido para gestão de farmácias municipais com distribuição controlada de:
1) Medicamentos para pacientes que fazem uso de remédios doados pelo sistema público.
2) Insumos distribuídos de uma central para unidades descentralizadas de saúde.

O software também faz:
- Gestão de pacientes de um município
- Gestão de funcionários
- Gestão de unidades de saúde
- Controle de estoques

Desenvolvido em VB.NET 2005 com banco de dados SQL Server Express 2005. Faz uso de Crystal Reports.
